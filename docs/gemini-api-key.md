# Gemini API 키 발급 방법

유튜브 지식 추출기에서 **Google Gemini** 를 LLM 제공자로 쓰려면 본인의 **Gemini API 키**가
필요합니다. 로컬에 Claude Code CLI 를 설치하지 않은 사용자는 이 방법을 권장합니다.

키 발급은 **무료**이며(구글 계정만 있으면 됨), 넉넉한 무료 사용량이 제공됩니다.

---

## 1. Google AI Studio 접속

브라우저에서 아래 주소를 엽니다.

👉 **https://aistudio.google.com/apikey**

구글 계정으로 로그인하라고 하면 로그인합니다(평소 쓰는 지메일/구글 계정이면 됩니다).

## 2. API 키 만들기

1. 페이지의 **"Create API key"(API 키 만들기)** 버튼을 누릅니다.
2. 프로젝트를 고르라고 하면, 기본으로 제안되는 프로젝트를 그대로 선택하거나
   **"Create API key in new project"(새 프로젝트에 만들기)** 를 누릅니다.
3. `AIza…` 로 시작하는 긴 문자열이 만들어집니다. 이것이 API 키입니다.

## 3. 키 복사

만들어진 키 옆의 **복사** 아이콘을 눌러 클립보드에 복사합니다.

> ⚠️ 이 키는 **비밀번호처럼 다뤄야 합니다.** 다른 사람에게 공유하거나 공개된 곳(깃허브,
> 채팅, 스크린샷)에 노출하지 마세요. 노출되면 AI Studio 에서 해당 키를 삭제(Revoke)하고
> 새로 만들면 됩니다.

## 4. 앱에 입력

1. 유튜브 지식 추출기를 실행합니다.
2. **고급 옵션** 을 펼칩니다.
3. **"LLM 제공자"** 를 **Google Gemini (API 키)** 로 바꿉니다.
4. 아래에 나타나는 **"Gemini API 키"** 칸에 복사한 키를 붙여넣고 **"키 저장"** 을 누릅니다.
5. 상태 문구가 **"Gemini API 키 설정됨 ✓"** 로 바뀌면 완료입니다.

키를 저장하면 앱이 자동으로 **사용 가능한 최신 모델 목록**을 불러옵니다(모델 드롭다운 옆
새로고침 버튼으로 다시 불러올 수도 있습니다). Google 이 새 모델을 출시하면 이 목록에
자동으로 나타납니다.

> 🔒 입력한 키는 **평문 설정 파일이 아니라 Windows 자격증명 관리자(OS 자격증명 저장소)** 에
> 안전하게 저장됩니다. 화면 입력칸은 저장 후 자동으로 비워집니다.

## 모델 선택 가이드

모델 드롭다운은 **"항상 최신" 별칭**과 **실제 사용 가능한 구체 버전**을 함께 보여줍니다.

| 항목 | 설명 |
| --- | --- |
| **Gemini Flash (최신)** | 항상 최신 Flash 로 라우팅(`gemini-flash-latest`). 기본값 · 빠르고 저렴 |
| **Gemini Pro (최신)** | 항상 최신 Pro(`gemini-pro-latest`) · 더 높은 품질(느리고 비쌈) |
| **Gemini Flash-Lite (최신)** | 항상 최신 Flash-Lite(`gemini-flash-lite-latest`) · 최저가 |
| **Gemini 2.5 Flash / Pro (안정)** | 버전을 고정하고 싶을 때 쓰는 안정 핀 |
| 구체 최신 버전 (예: `gemini-3.6-flash`) | **키 저장 후** API 에서 실제 목록을 불러와 자동으로 추가됨 |

- 버전 번호가 자주 바뀌므로(3.5 → 3.6 …) **"(최신)" 별칭**을 쓰면 새 모델이 나와도 자동으로
  최신을 사용합니다. 별칭 옆에는 현재 가리키는 구체 모델명이 함께 표시됩니다
  (예: `Gemini Flash (최신) → gemini-3.6-flash`).
- 모델 드롭다운 옆 **새로고침(⟳)** 버튼을 누르면 사용 가능한 최신 모델 목록을 다시 불러옵니다
  (API 키가 저장돼 있어야 합니다).
- 목록에 없는 모델도 모델 칸에 **모델 ID 를 직접 입력**해서 쓸 수 있습니다.

## 자주 묻는 질문

**Q. 구글 로그인만으로는 못 쓰나요?**
소비자용 Gemini API 는 로그인(OAuth)이 아니라 **API 키** 방식입니다. 위 절차로 키를 한 번
발급해 두면 계속 쓸 수 있습니다.

**Q. 키를 바꾸고 싶어요.**
AI Studio 에서 새 키를 만든 뒤, 앱에서 새 키를 입력하고 다시 **"키 저장"** 을 누르면 됩니다.

**Q. Claude 로도 쓸 수 있나요?**
네. Claude Code CLI(로컬 설치 + `claude login`)가 있으면 "LLM 제공자"를 Claude 로 두고
그대로 사용할 수 있습니다.
